ここからはcssの説明

cssの基礎の復習

  • cssはWebサイトの見た目を作る

  • cssを読み込むにはhtmlのhead内に

    と書けばよかった

  • cssの基本形は
    セレクタ {プロパティ: 値;}
    例えば

    • 
              p {
                color: red;
              }

色々なセレクタ

  • ユニバーサルセレクタ
    → 要素全部を指定
    書き方: *

  • 要素セレクタ
    → 各要素を指定
    例: p, h1, div

  • idセレクタ
    → 各idを指定
    書き方: #id名

  • classセレクタ
    → 各classを指定
    書き方: .class名

複数のプロパティを書くには

セレクタ {プロパティ1: 値1; プロパティ2: 値2;}
とするだけ

例えば


            p {
              color: red;
              background-color: blue;
            }

セレクタ応用編

2つのセレクタaとbを考える
例えば, divとpとか

  • a, b {スタイル}
    aにもbにもスタイルが適用される

  • a b {スタイル}
    aの下にあるbにスタイルが適用される

  • a > b {スタイル}
    aの直下にあるbにスタイルが適用される

  • a + b {スタイル}
    aの次に来るbにスタイルが適用される

  • ab {スタイル}
    aであってbでもあるものにスタイルが適用される
